-------------------------------------------------------------------------------

*package名稱要求全部小寫

-------------------------------------------------------------------------------

服務類別
    服務類別的package為 service.服務名稱
    例如:
        服務名稱為 Leaderboard
        服務類別的package為 service.leaderboard
        
-------------------------------------------------------------------------------

封包處理類別
    封包處理類別package為 service.服務名稱.processor
    例如:
        服務名稱為 Leaderboard
        封包處理類別的package為 service.leaderboard.processor

-------------------------------------------------------------------------------

設定類別
    設定類別的package為 service.服務名稱.config
    例如:
        服務名稱為 Leaderboard
        設定類別的package為 service.leaderboard.config
        
-------------------------------------------------------------------------------

資料類別 *用於存放從資料庫取出的資料
    資料類別的package為 service.服務名稱.data
    例如:
        服務名稱為 Leaderboard
        資料類別的package為 service.leaderboard.data
        
-------------------------------------------------------------------------------

資料來源類別 *用於執行資料庫存取動作
    資料來源類別的package為 service.服務名稱.dao
    例如:
        服務名稱為 Leaderboard
        資料來源類別的package為 service.leaderboard.dao
        
-------------------------------------------------------------------------------

樣板類別 *用於存放從樣板來源取出的資料
    樣板類別的package為 service.服務名稱.template
    例如:
        服務名稱為 Leaderboard
        樣板類別的package為 service.leaderboard.template
        
-------------------------------------------------------------------------------

樣板來源類別 *的類別執行樣板存取動作
    樣板來源類別的package為 service.服務名稱.tao
    例如:
        服務名稱為 Leaderboard
        樣板來源類別的package為 service.leaderboard.tao
        
-------------------------------------------------------------------------------

工具類別
    工具類別的package為 service.服務名稱.util
    例如:
        服務名稱為 Leaderboard
        工具類別的package為 service.leaderboard.util
        
-------------------------------------------------------------------------------

雜項類別
    雜項類別的package為 service.服務名稱.model
    例如:
        服務名稱為 Leaderboard
        雜項類別的package為 service.leaderboard.model
        
-------------------------------------------------------------------------------
